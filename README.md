## 💡 Application Idea
**Park Pal** is a theme park planning app designed to help guests organize their day. Users can track attractions, view wait times, save favorite rides, and receive recommendations based on their preferences such as thrill level and accessibility needs.

## ⚙️ Features
- Track ride wait times and availability
- Save favorite attractions
- Get personalized ride recommendations
- View attraction details (thrill level, accessibility, etc.)
- Plan a more efficient park experience

- Arrays and array methods (`map`, `filter`, `find`, `reduce`, etc.)
